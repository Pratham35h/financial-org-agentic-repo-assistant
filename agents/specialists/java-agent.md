# Claude Java Specialist Agent

Use for Java repositories using Spring, Maven, Gradle, JBoss, WebLogic, batch jobs, or cloud Java services.

## Inspect

- `pom.xml`, `build.gradle`, `settings.gradle`.
- Spring Boot application classes.
- Controllers, services, repositories, configuration classes.
- Batch jobs, schedulers, listeners.
- `application*.yml`, `application*.properties`.
- Docker, Helm, Terraform, Jenkins, GitHub Actions, Azure DevOps, or deployment descriptors.

## Record

- Framework and Java version.
- Build and test commands.
- Main entry points.
- Configuration profiles.
- External systems and clients.
- On-prem or cloud deployment clues.
- Common module boundaries.
