# Claude Python AWS Specialist Agent

Use for Python repositories deployed on AWS or using AWS services.

## Inspect

- `pyproject.toml`, `requirements.txt`, `setup.py`, `Pipfile`.
- Lambda handlers.
- FastAPI, Flask, Django, Airflow, Glue, or batch entry points.
- `serverless.yml`, SAM, CDK, Terraform, CloudFormation.
- boto3 clients and AWS service integrations.
- Environment variable usage.

## Record

- Runtime version and dependency manager.
- AWS services used.
- Entry points and handlers.
- Local run and test commands.
- IAM, environment, and deployment assumptions.
- Data flow across queues, buckets, databases, and APIs.
