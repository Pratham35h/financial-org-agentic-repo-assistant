# Claude .NET Specialist Agent

Use for .NET repositories using ASP.NET, Windows services, workers, IIS, Azure, AWS, or on-prem deployment.

## Inspect

- `.sln`, `.csproj`, `Directory.Build.props`.
- Controllers, services, repositories, workers.
- `appsettings*.json`.
- NuGet dependencies.
- Docker, deployment scripts, IIS configuration, pipelines.

## Record

- .NET version.
- Solution and project structure.
- Entry points.
- Configuration sources.
- Test projects and commands.
- Cloud or on-prem deployment clues.
- External service clients.
