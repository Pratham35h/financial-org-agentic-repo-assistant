# Stack Detection Playbook

Use these signals to detect repository type.

## Java

- `pom.xml`
- `build.gradle`
- `settings.gradle`
- `src/main/java`
- `application.yml`
- Spring annotations
- JAR/WAR/EAR packaging

## Python AWS

- `requirements.txt`
- `pyproject.toml`
- `setup.py`
- `serverless.yml`
- `template.yaml`
- `cdk.json`
- `boto3`
- Lambda handlers
- Glue or Airflow code

## .NET

- `.sln`
- `.csproj`
- `Program.cs`
- `Startup.cs`
- `appsettings.json`
- `packages.config`

## COBOL Mainframe

- `.cbl`, `.cob`, `.cpy`, `.jcl`
- Copybooks
- JCL PROCs
- DB2 SQL blocks
- VSAM dataset references
- CICS transaction references
