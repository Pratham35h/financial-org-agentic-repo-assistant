# Investment Operations Skill

Load this skill for trades, positions, corporate actions, holdings, reference data, security master, portfolio setup, or post-trade processing.

## Primary Concerns

- Trade lifecycle, amendments, cancellations, corrections, and allocations.
- Position building from transactions, holdings files, and corporate actions.
- Security identifiers and golden-source precedence.
- Custodian/admin position and transaction feeds from State Street, BNY, Citi, SS&C, DTCC, and SWIFT-derived sources.
- Market/reference data from Bloomberg, Morningstar, Refinitiv/LSEG, S&P, ICE, MSCI, Moody's, and Fitch.

## Ask Before Changing

- Identifier precedence, book-of-record ownership, or corporate action treatment is unclear.
- The change affects settled positions, accounting positions, or downstream reporting.
